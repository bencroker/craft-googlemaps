# Location Type

:::tip Limited Usage
Only appears within the [`GoogleMaps_Geometry`](/graphql/geometry-type/) type.
:::

```graphql
type GoogleMaps_Location {
  lat: Float
  lng: Float
}
```

### `lat`

_Float_ - Latitude of the place.

### `lng`

_Float_ - Longitude of the place.
